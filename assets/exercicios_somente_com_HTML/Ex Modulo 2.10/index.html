Uso da tag canvas

<canvas> é um elemento HTML que cria um quado bitmap na página que pode ser usado para reprenetação gráfica usando a linguagem de "script" (normalmente JavaScript ). Isto pode ser usado, por exemplo, para desenhar gráficos, fazer composições de fotos ou animações simples (e não tão simples ).
