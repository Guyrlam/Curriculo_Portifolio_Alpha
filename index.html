Formulário simples

Neste exercício, construímos um formulário simples, com o uso de tags como <form>; <label>; <input>; e <buttom>. Essas, auxiliadas por seus respectivos atributos, acionam os widgets dos navegadores, que permitem a interação entre um usuário e um website ou aplicativo. Na maior parte do tempo, os dados captados nessas interações são enviados para o servidor da web, mas a página também pode interceptar para usá-los por conta própria.

Formulário completo

Neste exercício, além de utilizarmos os elementos básicos apresentados no formulário acima, acrescentamos novas tags, atributos e, principalmente, valores os vinculados ao atributo 'type' da tag 'input', como 'checkbox', 'radio', 'text', 'email', 'range', 'radio', 'date', etc. Além disso, foram utilizadas as boas práticas de posicionamento e agrupamento por meio das tags <fieldset>, para agrupar tags; e <optgroup> para agrupar as tags <option> dentro da <select>. Assim como a definição correta de campos obrigatórios e navegação por meio da tecla tab.

Canvas

Neste exercício, utilizamos a tag <canvas>, que cria um quadro bitmap na página que pode ser usado para reprenetação gráfica usando a linguagem de "script" (normalmente JavaScript ). Isso pode ser usado, por exemplo, para desenhar gráficos, fazer composições de fotos ou animações simples (e não tão simples ).
